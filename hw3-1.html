Animal Name Goes Here

Use internal (or external) JavaScript to implement the features below:

  • Display the image of the animal corresponding to the button the user clicks on.
  • Change the color of the image border based on the animal displayed: yellow for Shepherd, red for Penguin, green for Snow Leopard and lime for Lynx. If my color choices are poor, you can pick your own colors as long as it is different and unique for each animal image.
  • Change the heading, "Animal Name Goes Here" to the type of animal shown.
  • Change the background color of the button that the user clicks on to Silver. Only the button the user has clicked on must be Silver. All others must be White.
  • Write another function to enlarge the image by 75% when the user mouses-over it and return it to normal size when the user mouses-out of the image.
  • If you use a single function for all of the above functionality, you will get extra 5 bonus points. If you cannot, please use as many functions as needed.
